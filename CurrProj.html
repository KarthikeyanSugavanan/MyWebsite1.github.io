IoT and AI based Eye tracking aiming device


IOT + AI

We plan to do a Military based application project for this semester using AI and IoT. The system will be controlled using eye movements, the user can control the aiming dot using his/her eye movement using IoT and Eye-tracking software. The environment captured by the camera will act as the input for the AI software, which is trained to distinguish between different color objects.

The User can shoot the intended target by blinking the eye, the AI system will verify if the intended object is a Friend or Enemy based on color of the object, if Enemy the AI will shoot the target whereas if the target is a Friend, the AI will disarm and will not shoot.
